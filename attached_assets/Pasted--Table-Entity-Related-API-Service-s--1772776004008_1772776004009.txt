### PostgreSQL Extensions & Configuration Notes

| Feature                      | Detail                                                                                                          |
| ---------------------------- | --------------------------------------------------------------------------------------------------------------- |
| **pgvector**           | Required for `document_chunks.embedding` column. Install via `CREATE EXTENSION vector;`                     |
| **uuid-ossp**          | UUID generation for all primary keys.`CREATE EXTENSION IF NOT EXISTS "uuid-ossp";`                            |
| **pg_trgm**            | Trigram-based fuzzy text search for question bank and course search.`CREATE EXTENSION pg_trgm;`               |
| **Indexes**            | GIN indexes on all JSONB columns, GiST index on vector column, B-tree on all FK columns and status/enum fields. |
| **Partitioning**       | `audit_log` and `notifications` tables partitioned by `created_at` (monthly range) for query performance. |
| **Row-Level Security** | Enabled on `users`, `enrollments`, `exam_attempts` for multi-tenant data isolation where applicable.      |
| **Migration Tool**     | Alembic with async SQLAlchemy support. All schema changes versioned and reversible.                             |
| **Connection Pooling** | asyncpg driver with SQLAlchemy async engine. PgBouncer recommended for production connection pooling.           |
