# Backend API Services

> **Tech Stack:** Python 3.11+ · FastAPI · PostgreSQL · SQLAlchemy (async) · Celery + Redis · LangChain · pgvector

---

## 1. Authentication Service

| | |
|---|---|
| **Screen** | Login & Role Access |
| **Method** | POST |
| **Endpoint** | `/api/v1/auth/login` |
| **Description** | Validates credentials (email/password or SSO token), issues JWT access & refresh tokens, returns user role for client-side redirection. |
| **Tech** | FastAPI + OAuth2PasswordBearer, python-jose for JWT, passlib[bcrypt] for hashing |

---

| | |
